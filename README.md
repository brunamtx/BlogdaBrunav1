# Blog da Bruna

Um blog na estética anos 2000, feito com **HTML, CSS e JavaScript**

Ele foi idealizado (e ainda sujeito à alterações) como um lugar pessoal de escrita, e também como forma de estudo.

Nele, nós temos: 
- menu lateral
- gifs
- marquee
- contador de visitas

Tecnologias usadas: HTML5, CSS3, JavaScript (vanilla).
